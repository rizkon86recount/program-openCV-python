untuk menjalankan program program opencv diatas silahkan instal berikut ini:
1. Pastikan Anda telah menginstal Python di komputer Anda. 
Jika belum, Anda dapat mengunduh dan menginstalnya dari situs web resmi Python: https://www.python.org/downloads/
2. Instal modul pygame dan OpenCV jika Anda belum melakukannya. Anda dapat melakukannya dengan perintah berikut: pip install pygame opencv-python
3. inst mediapipe, perintahnya seperti berikut ini "pip install mediapipe"
